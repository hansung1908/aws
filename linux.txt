리눅스 명령어
clear - shell 화면 초기화
pwd - 현재 경로 위치
cd (경로 위치) - 해당 경로 위치로 이동(cd .. - 한단계 상위 경로로 이동, /는 최상위 경로)
절대경로 - /home(최상위 경로부터 이동), 상대경로 - ubuntu(현재 폴더부터 이동)
ls - 현재 폴더의 파일,폴더 상태 표시(흰색 - 파일, 파란색 - 폴더, 초록색 - 실행 파일) (ls -l - 자세히 보기, 작대기 - 파일, d - 폴더)

(명령어) --help - 해당 명령어의 모든 사용법을 알려줌
ls -a, --all - 현재 폴더의 모든파일(숨김파일, 기본파일, 기본폴더)를 보여줌
ls -al - 현재 폴더의 모든파일을 자세히 보기
mkdir (폴더 이름) - 폴더 생성
touch (파일 이름) - 파일 생성
rm (파일 이름)- 파일 삭제
rm -r, --recursive (폴더 이름) - 삭제하려는 폴더 안에 내용물도 같이 삭제
ctrl + c(^c) - 취소

cp (파일 이름) (복사본 이름) - 해당 파일을 복사하여 생성
mv (파일 이름) (폴더 이름) - 파일을 폴더로 이동(절대 경로로 이동할 경우 폴더 이름에 경로 작성)
mv (파일 이름) (새로운 파일 이름) - 새로운 파일 이름으로 변경(폴더 이동 시 폴더 이름뒤에 /(새로운 파일 이름)을 붙혀서 변경도 같이 가능)
ln -s - 소프트 링크라고 함, 바로가기 파일 생성(바로가기 파일 -> 실제 파일)
bin 폴더 - 실행파일이 모여있는 곳
